binary_test问题修复（Phase 1恢复）
问题现象
运行binary_tset.py报错：
    ImportError: attempted relative import with no known parent package

问题定位
    binary_test以脚本方式运行
    binary_decision.py使用相对导入：
        from .ai_infer import Detection
    导入机制冲突：
        脚本运行-不存在package上下文
        相对导入-依赖package结构

结论
    该问题属于模块导入方式不一致导致的工程集成问题，而非算法或模型问题

解决方案
    统一为脚本级绝对导入：
        from ai_infer import Detection

    验证结果
        binary_test.py成功运行
        推理与二元判定逻辑正常执行
        输出结果可生成
