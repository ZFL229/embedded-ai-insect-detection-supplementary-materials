# Supplementary Materials for Embedded AI Insect Detection System

This repository contains supplementary materials for the thesis project "Design and Implementation of an Embedded AI Vision System".

The main source code is stored in a separate repository:
https://github.com/ZFL229/embedded-ai-insect-detection-system.git

## Structure

- `Appendix_B_Supplementary_Image_Data/`
  - Supplementary image data used or generated during AI validation, camera integration, and final system testing.

- `Appendix_C_System_Running_Logs/`
  - System running logs, communication logs, error-injection logs, robustness-test logs, and final validation logs.

These files are provided for traceability and are not required to understand the main thesis content.
